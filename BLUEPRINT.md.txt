# BLUEPRINT.md
# OceanFresh — Full Product Development Blueprint

Version: 1.0
Framework: Next.js + Tailwind CSS + TypeScript
AI IDE Target: Antigravity IDE
Secondary Support: Cursor / Windsurf / Claude Code

====================================================
PROJECT OVERVIEW
====================================================

OceanFresh is a modern multi-role seafood commerce platform.

The system includes:

1. Customer Marketplace
2. Seller Dashboard
3. Admin Dashboard

The application must feel:
- Premium
- Enterprise-grade
- Mobile-first
- Highly scalable
- AI-development friendly
- Component-driven
- Fully reusable

IMPORTANT:
This blueprint is the SINGLE SOURCE OF TRUTH for Antigravity IDE.

Antigravity MUST:
- Follow this architecture exactly
- Preserve design consistency
- Avoid redesigning UI
- Reuse components everywhere
- Maintain strict coding standards

====================================================
CORE DEVELOPMENT PRINCIPLES
====================================================

NEVER:
- Randomly redesign UI
- Create inconsistent spacing
- Create duplicate components
- Change design tokens
- Use inline styles
- Use inconsistent Tailwind utilities
- Create random page layouts

ALWAYS:
- Reuse components
- Reuse spacing system
- Use design tokens
- Preserve responsiveness
- Use TypeScript strictly
- Use modular architecture
- Maintain scalable folder structure
