OceanFresh — Customer Marketplace UI
Modern Premium Seafood Commerce Experience

====================================================
OVERVIEW
====================================================

OceanFresh Customer UI is designed as a modern premium seafood shopping experience inspired by:

- Apple Store
- Airbnb
- Uber Eats
- Amazon Fresh
- Deliveroo
- Modern D2C brands

Unlike the Admin and Seller dashboards, the Customer UI should feel:
- More emotional
- More visual
- More immersive
- More product-focused
- More mobile-native
- More lifestyle-oriented

The customer experience must feel:
- Fresh
- Premium
- Fast
- Trustworthy
- Ocean-inspired
- Clean and modern

IMPORTANT:
Customer UI should still follow the SAME core design system as the dashboards but with:
- Slightly softer visuals
- More gradients
- Larger product imagery
- More whitespace
- More rounded surfaces
- More visual storytelling
